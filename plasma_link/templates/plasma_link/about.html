About

What is PlasmaLink?/h5>

PlasmaLink helps connect plasma donors and patients in need of convalescent plasma therapy for COVID treatment. PlasmaLink was made in wake of India's devastating second COVID wave.

PlasmaLink is open source under MIT License. View project on GitHub . Feel free to contribute by opening a pull request.

How can I provide feedback/report an error?
If you want to talk about the code, you may also open an issue on GitHub .
